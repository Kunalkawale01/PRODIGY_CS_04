Create a basic keylogger program that records and logs keystrokes. Focus on logging the keys pressed and saving them to a file. Note: Ethical considerations and permissions are crucial for projects involving keyloggers.

# Simple Educational Keylogger

## Project Overview
This project is a **basic, consent-based keylogger** developed **strictly for educational and cybersecurity learning purposes**.  
It demonstrates how keyboard input can be captured at the operating system level using Python.

**This project is NOT intended for malicious use.**

---

## Ethical & Legal Disclaimer
This software is created **only for learning and demonstration purposes**.

Use it **only** on:
- Systems you own  
- Systems where you have **explicit written permission**

Unauthorized use of keyloggers is **illegal and unethical**.

The author takes **no responsibility** for misuse of this project.

---

## Features
- Logs all keyboard key presses
- Stores logs in a text file (`keylog.txt`)
- Displays console messages (no stealth behavior)
- Safely stops logging when **ESC** is pressed
- Lightweight and easy to understand
